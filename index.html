Projects

CalorieBuddy - Calorie Deficit Recipe Webapp

Developed a fully responsive, dynamic web application that generates personalized meal plans based on user-defined calorie goals and dietary restrictions. This idea was inspired by my grandmother, who told me she wanted to track her calories but didn't know how. To build this, I integrated the Spoonacular API to fetch real-time recipe data, including nutritional information and preparation time. I also designed an intuitive UI with React.js, ensuring seamless user interaction and optimal performance across devices.

Petpal - A pet adoption website

Built a website that helps users find the perfect pet to adopt, based on factors such as location, breed and age. Giftit - Mobile Application Design

This was a project based on user-centred design of interactive systems. Collaborated on a 5 person team to create an application that solves the problem of picking the right gift for a recipient. We first used interviews and questionaires to better understand our primary stakeholders and the problem. From this we were able to create a low fidelity prototype and conduct extensive usability testing for refinement. We used a multiple iteration process and were able to come up with a final high fidelity prototype using Figma. We then presented our findings.
